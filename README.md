# Voidless Game Project

This repository hosts the source for the Voidless Game project's website. The site is built with [Jekyll](https://jekyllrb.com/) and deployed through GitHub Pages.

## Local Development

To build the site locally:

1. Install dependencies:
   ```sh
   bundle install
   ```
2. Build the site:
   ```sh
   bundle exec jekyll build
   ```
   The generated site will appear in the `_site/` directory.

## Deployment

This repository is published using GitHub Pages. Changes pushed to the `main` branch are automatically built and deployed by GitHub Pages.
